# 📥 Установка

## 1. Скачивание

Скачайте последнюю версию из раздела **Releases**.

---

## 2. Распаковка

Распакуйте архив в отдельную папку.

> Рекомендуется использовать путь без русских букв и пробелов.

### Пример:

```text
C:\Users\%Username%\Desktop
```

---

## 3. Отключение антивируса

> [!WARNING]
> Некоторые антивирусы могут ошибочно определять `zapret` как подозрительное ПО.
>
> Это связано с использованием сетевых фильтров и драйвера `WinDivert`.
>
> Перед запуском рекомендуется:
>
> - временно отключить антивирус,
> - добавить папку программы в исключения Windows Defender.

---

## 4. Запуск

Запустите нужный `.exe` файл **от имени администратора**.

### Примеры:

```text
▶️ YouTube zapret.exe
🎮 Discord Zapret.exe
📊 Telegram Zapret.exe
🚀 Старт всех конфигов.exe
```

После запуска дождитесь открытия окна консоли.

# ❓ FAQ

<details>
<summary><b>После запуска сайты всё равно не работают</b></summary>

Попробуйте:

- перезапустить браузер,
- закрыть Discord/Telegram полностью,
- очистить DNS-кэш:

```bat
ipconfig /flushdns
```

</details>

<details>
<summary><b>Работает ли вместе с VPN?</b></summary>

Нет. Для корректной работы рекомендуется полностью отключить VPN и прокси.

</details>

<details>
<summary><b>Как удалить программу?</b></summary>

Просто закройте окно программы.

Дополнительные службы и фоновые процессы не устанавливаются.

</details>

<details>
<summary><b>Почему антивирус жалуется?</b></summary>

Программа использует сетевые драйверы и методы фильтрации трафика (`WinDivert`), из-за чего некоторые антивирусы могут выдавать ложные предупреждения.

Это нормальная реакция для подобных инструментов.

</details>
